### 📌 Project Description

This project is about designing and building a custom Printed Circuit Board (PCB) using EasyEDA. Instead of using a breadboard or loose wires, the circuit is turned into a clean, compact, and reliable board. This makes the circuit easier to use, more durable, and looks more professional.

---

### 🛠️ Steps I Took

1. Started by choosing a simple circuit and listing the components needed.
2. Created the schematic in EasyEDA by placing and connecting components.
3. Converted the schematic into a PCB layout.
4. Arranged components neatly and routed the connections.
5. Checked for errors and exported the design files (Gerber files).
6. Got the PCB manufactured and soldered all the components.
7. Tested the final board to make sure everything works properly.

---

### 🧾 Bill of Materials (BOM)

* Resistors
* Capacitors
* LED(s)

* Connectors
* Custom PCB
* Wiring
